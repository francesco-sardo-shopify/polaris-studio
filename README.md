# Polaris Studio

- `pnpm install`
- `cp .env.example .env.local`
- Generate a personal key here https://openai-proxy.shopify.io/
- Add it to your env local: OPENAI_LIKE_API_KEY={KEY}
- `pnpm dev`
- Visit `http://localhost:5173/`
